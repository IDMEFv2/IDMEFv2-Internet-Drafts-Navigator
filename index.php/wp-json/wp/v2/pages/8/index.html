About us<\/h1>\n\n\n\n

The IDMEFv2 Task Force is an international consortium working on the definition of a universal format for incident detection. The IDMEFv2 format is based on JSON and proposes classes and attributes for a structured description of physical and cyber incident and intrusion. <\/p>\n\n\n\n


The IDMEFv2 format is described in an official IETF V00 Draft. The V01 version is being finalized and from now on the consortium is open to external contribution in order to create a community of interest before entering a standard track process at IETF.<\/p>\n\n\n\n

Joining the community is FREE<\/a><\/strong><\/p>\n\n\n\n


Documentation is FREE<\/a><\/strong><\/p>\n\n\n\n


Code is FREE (Open-Source)<\/a><\/strong><\/p>\n\n\n\n


This website host all the available information and tools around IDMEFv2 so that the community can understand, test and appropriate the format and join the IDMEFv2 community.<\/p>\n\n\n\n
